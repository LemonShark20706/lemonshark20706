## A little introduction.:
My name is Adam Olah. I'm a young developer currently learning programming as a profession. As you can see, I have a strong interest in **Python**, *geo data science*, and automation.  
I have several projects that explore these topics in different ways.

The first category is **Manual Only Scripts**, which work well on their own but may struggle when interacting with other scripts or projects.

The second category is **Automata Only Scripts**, which require only a *data source* file and need no further user input. This type can work as a standalone script, but it may still encounter issues when integrated into larger projects.

The third category is **Semi‑Automata and Manual Scripts**, which also use *data source* files. If the provided data is incomplete or incorrect, the user must supply the missing information. This type is the most likely to work well when integrated into other projects.

The fourth category is a **full automation approach**, combining multiple existing projects into a single automated workflow.

Most of my projects include both a GUI and a console version to support different user preferences.

I design my projects to work with a utils folder, allowing users to extend my base work and create their own custom processes if needed.
